HTML

  • The head element is the opening element where the metadata can be found. Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
  • The body elemnt represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.
  • The element that that holds all the children is known as the parent element.
  • A HTML attribute defines additional information about an element; such as language. They are always defined in the beginning elemnt, never the closing element.
