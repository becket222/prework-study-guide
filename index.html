Prework Study Guide

HTML

  • The head element is the opening element where the metadata can be found. Metadata is data about our webpage that the browser uses to effectively serve our webpage to users.
  • The body elemnt represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.
  • The element that that holds all the children is known as the parent element.
  • A HTML attribute defines additional information about an element; such as language. They are always defined in the beginning elemnt, never the closing element.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the elemnt or attribute to which the rules, or declarations, will apply.
  • Declarations contain: the CSS property we want to apply; the value of the property.
  • You can combine two selectors into a single CSS rule with a comma, allowing the declaration to affect both selectors.
  • The h1 and h2 rule assignment is a positioning property; it allows you to centre a title or heading on a webpage.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it; 'branch' is main or feature; 'name' is what you named said branch.
